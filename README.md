# Projeto Academia Techfit

Este repositório contém o desenvolvimento do projeto **Techfit**, uma aplicação voltada para gestão de academia. <br>
Abrange desde a concepção da base de dados até a implementação da lógica de negócio e interface. <br>
Projeto de conclusão de módulo, demonstrando competências em Full Stack e Modelagem de Dados.

---

## 📂 Estrutura do Repositório

O projeto está organizado para refletir as diferentes etapas do desenvolvimento:

* **`techfit/` (ou `techfit_php`)**: Contém o código-fonte da aplicação desenvolvido em **PHP**, focando na integração com o banco de dados e regras de negócio.
* **`ATV BRMODELO`**: Arquivos de modelagem de dados (DER/MER) criados no brModelo, garantindo uma estrutura de banco de dados normalizada.
* **`DOCUMENTAÇÃO`**: Toda a parte teórica e técnica do projeto, incluindo levantamento de requisitos e manuais.
* **`IMAGENS`**: Assets visuais e prints da interface do sistema em funcionamento.

---

## 🛠️ Tecnologias Utilizadas

* **PHP:** Linguagem server-side para processamento de dados.
* **SQL:** Para gerenciamento e consultas ao banco de dados.
* **brModelo:** Ferramenta de modelagem de dados.
* **HTML5 & CSS3:** Para a construção e estilização da interface do usuário.

---

## 📝 Funcionalidades Principais

* Gerenciamento de usuários/alunos.
* Painel Administrativo (conforme indicado pelo arquivo `senha adm`).
* loja de produtos.

---

## 🚀 Como executar o projeto

1. Clone o repositório.
2. Certifique-se de ter um ambiente com PHP e MySQL (como o **XAMPP**).
3. Importe o banco de dados (baseado nos arquivos da pasta `ATV BRMODELO`).
4. Mova a pasta do código para o diretório `htdocs` e acesse via `localhost`.
